# 🚑 Real-Time Generative AI Emergency Triage Assistant

## Problem
In emergency rooms and disaster zones, doctors must quickly analyze patient symptoms and large medical protocols. Reading hundreds of pages wastes critical time.

## Solution
This project builds a Generative AI triage assistant that reads medical documents and recommends the next best emergency action instantly.

## Features
- ⚡ Low latency emergency response
- 📄 Medical document analysis
- 🧠 AI-powered protocol recommendation
- 🏥 Hospital-style dashboard
- 📊 Vector search using FAISS

## Technology
- Python
- Streamlit
- Sentence Transformers
- FAISS Vector Search

## How to Run

Install dependencies

pip install -r requirements.txt

Run the application

streamlit run app.py

## Example Query

patient unconscious chest pain

## Output

Critical Emergency Detected  
Recommended protocol: Start CPR and alert cardiac team.
